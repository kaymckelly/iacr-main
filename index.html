Versions of IACR website refresh

The goal here is to make the IACR website mobile-friendly. The site would be revised with a new header, menu, and footer. These variations incorporate different features for the home page, and we can pick and choose for the home page. The images for conferences are subject to change, but there are various versions in different examples.

Version 1

  • Top row is upcoming events.
  • Four cards evenly distributed below events.
  • Menu is full width, header is not.

Version 2

Pages:
  • main page
  • Journal of Cryptology
  • Very compact header, dual level menu.
  • Search bar can be die
  • Horizontal event cards
  • Various color schemes

Version 3

Main page and Journal of Cryptology page
  • Added login button (do we need this?)
  • Narrow nav bar with search and autocomplete
  • Six cards on home page (community is redundant?).
  • Featured video card omits embedded player (no cookies!)

Version 4

  • A wide version. Note that only the main page would be this wide.
  • Menu has hover dropdowns
  • Icons on news items
  • Publication card with images for publications (open access!)

Version 5

Main page
  • Hover menus on full-width nav
  • Shows six events above the fold with vertical event cards
  • Cards in varying sizes

Version 6

Main page.
  • Large header with large logo. Could use wordmark there.
  • Menu bar is aligned to sides
  • Horizontal event cards
  • Map card
