Versions of IACR website refresh

Round two demos.

There are now two menu structures to choose from. Each menu structure has several possible color schemes and several possible footers. The two menu structures are referred to as v2 and v3. They no longer show the home page, but rather the appearance of a "generic" page.

Version 4 (release candidate)

This version was culled from comments in the Feb. 1 meeting.

OLDER VERSIONS BELOW

Version v2

  • A large header with the IACR name always on the page.
  • Various color schemes to select from (screenshot from one).

The mobile version menu structure has search at the bottom of the menu, but auto-scrolls to the top when it receives focus. This still needs work on iPhones, because they do not support focus/blur events in a standard way.

Version v3

  • A sticky top nav that remains on screen but is very compact. As the user scrolls down the page, the top nav remains visible.
  • The name "International Association for Cryptologic Research" would only appear on the header of the main page.
  • Various color schemes to select from (screenshot from one).

The mobile version menu structure has search being opened separately.

NOTE: There is also a search demo, but search is now integrated anyway. There is also a list of all available images here
